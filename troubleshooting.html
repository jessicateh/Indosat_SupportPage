    Troubleshooting common issues


    Code Message Description Solution

    -700
    • Invalid staff PIN
    • The user has entered an incorrect PIN or this device has been blocked or removed by an administrator.
    • Check your PIN and try again.
    • Note that if the wrong PIN is entered 3 times, the device will be blocked and a new PIN must be generated.
    • If you cannot remember your PIN, you can reset your staff PIN by going to Settings > Staff management in the app and selecting reset staff PIN. A new PIN will be emailed to you.
    • If the device has been blocked due to wrong PIN entry, contact support.

    -844
    • Connection Timeout. Please check your connectivity and try again.
    • The app was unable to connect to the server. It tried but after a certain time, it gave up.
    • Check that you are connected to the Internet, make sure you are on a stable connection and try again.

    -847
    • Reversal from mobile successful.
    • The app waited but not receive a response from the server, probably because of poor connectivity, and so it has requested that the transaction be reversed.

      The server has reversed the transaction and no charge has been made to the customer's card.
    • Check that you are connected to the Internet, make sure you are on a stable connection and try again.

    -928
    • Reversal has timed out. Please try again. If the problem persists, please contact support.
    • The app waited but not receive a response from the server, probably because of poor connectivity, and so it has requested that the transaction be reversed.

      However, the connection is still down or unstable and the app cannot communicate with the server.
    • Check that you are connected to the Internet, make sure you are on a stable connection and try again.

    -955
    • Failed to decrypt credit card information. Please check that the reader is supported or contact support.
    • The server did not recognise the encrypted message from the reader. This may be because the reader is not supported or because the reader has not been injected with the correct key.
    • If the Setting is available to you, go to app Settings > Key injection and inject the reader. This will update the key in the reader.

      You may need the Merchant PIN to access this setting. Your manager or mPOS administrator will have this PIN.

      After injecting the key, if the error recurs, please contact support directly.

    -960
    • Unknown card type. Please try a different card.
    • The card is not recognised by the system. This may be because the card issuer is not supported.
    • Ask the customer to try a different card.

      If you expect this card to work, there may be a fault with the card reader, in which case please contact support.

    -972
    • Card type not allowed due to provider BIN restriction. Please contact support.
    • The card is not allowed by the system.
    • Ask the customer to try a different card.

      If you expect this card to work, please contact support.

    -973
    • Card type not allowed due to merchant BIN restriction. Please contact support.
    • The card is not allowed by the system.
    • Ask the customer to try a different card.

      If you expect this card to work, please contact support.
