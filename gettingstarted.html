      Getting started

      Merchant portal

      Before using D-Pay application in your device, we recommend you to do a few settings in the portal first. These settings will provide you necessary information that you’ll need to key in for the application.

      Login to portal

      Upon login to the portal, you’ll be asked for username and password. These details can be found in the welcome mail sent to you.

      Create staff

      Your staff will have his/her own particular ID. To create staff for your device, click on “Staff and Devices” located in the sidemenu. Click on “Create new staff” and fill in the details in the form. Generate the Staff PIN. Staff will use this PIN if D-Pay application asks for identification before making transactions. Click "Submit", you'll be prompted a "Success!" message with staff name appears on the list.
      1. Steps to create new staff
      2. Form to create staff
      3. Prompting "Success" message with staff name appears in the list

      Create device and activation code

      After successfully creating staff, click on the “Eye” icon beside staff name and proceed with “Create new device”. Fill in the Device name. If you want to send the activation code via email, select “Email” in "User activation code via". Click "Submit" and your activation code will be generated. Use this code to activate D-Pay application in your device.

      2. Create new device
      3. Form to create new device

      Done? Awesome, let’s get things going with your D-Pay mobile application.
